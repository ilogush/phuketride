Обновлённые правила для AI-агента PhuketRide — система управления арендой автомобилей:
1. Отвечать кратко и по делу.
2. Все рассуждения и ответы — на русском языке.
3. Использовать ТОЛЬКО удалённую базу данных Cloudflare D1 (флаг --remote в wrangler).
   - ЗАПРЕЩЕНО использовать локальные SQLite файлы (.wrangler/state/v3/d1/*.sqlite).
   - Все операции с БД должны выполняться на production базе через wrangler d1 execute <db-name> --remote.
   - Для локальной разработки: npm run dev = wrangler dev --remote (прямое подключение к Cloudflare D1).
   - Локальные БД должны быть удалены: rm -rf .wrangler/state.
4. Самостоятельно формировать и выполнять SQL-запросы при необходимости.
5. Удалять временные файлы и служебные данные после использования.
6. Не создавать .md файлы без прямого разрешения.
7. Минимизировать лишние действия и операции вне запроса.
8. Ошибки валидации и действия отображать только в toast-уведомлениях, без дублирования.
9. Все иконки в проекте брать строго с https://heroicons.com.
10. Компоненты админки должны находиться только в `app/components/dashboard`, компоненты клиентской части — только в `app/components/public`; пересечение и переиспользование между этими папками запрещено.
11. Для локального сервера использовать фиксированный порт `8080`: порт не менять. При необходимости выполнять только очистку кэша и перезапуск сервера на том же порту.
