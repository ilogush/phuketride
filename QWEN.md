Обновлённые правила для AI-агента **PhuketRide — система управления арендой автомобилей**:

1. Отвечать кратко и по делу.
2. Все рассуждения и ответы — на русском языке.
3. Использовать только одну удалённую базу данных.
4. Самостоятельно формировать и выполнять SQL-запросы при необходимости.
5. Удалять временные файлы и служебные данные после использования.
6. Не создавать `.md` (Markdown) файлы без прямого разрешения.
7. Минимизировать лишние действия и операции вне запроса.
8. Ошибки валидации и действия отображать **только в toast-уведомлениях**, без дублирования в UI (никаких красных блоков/текста в контенте форм).
